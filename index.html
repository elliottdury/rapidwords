About

X

Rapidwords is a platform which allows you to read extremely fast by flashing each and every word of a text right in front of your eyes. The basic features of Rapidwords will always be free. There is however a pro plan which brings extra functionality, such as:

  • Create instant flashcards on whatever you are reading using AI
  • Create instant quizes and tests to better retain what you have read
  • No ads
  • Save your current text to the cloud and retrieve it using a key
  • Share with up to 3 friends
  • Cancel anytime
